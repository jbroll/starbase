
### `grestable` - table driven general regular expression substitution.

SYNOPSYS
--------

```
    `grestable` [-i *ifile*] [-o *ofile*] *table* *fromcol* *tocol* 
```

DESCRIPTION
-----------

Use sed to replace all of the regular expressions in *fromcol* with the 
strings in *tocol*.
