
NAME
====

`trantable` - table driven string substitution.

SYNOPSYS
========

```
    `trantable` [-i *ifile*] [-o *ofile*] *table* *fromcol* *tocol* 
```

DESCRIPTION
===========

Use m4 to replace all of the strings in *fromcol* with the 
strings in *tocol*.
