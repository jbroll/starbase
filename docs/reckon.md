

reckon - execute a block of awk program statements.


SYNOPSYS
--------

```
        `reckon` '*statements*'
```

DESCRIPTION
-----------

`reckon` allows the execution of a block of awk statements.  It evaluates its
arguments as a single awk language block and exits.

EXAMPLES
--------

```
        reckon 'print "X"'
        X
```
